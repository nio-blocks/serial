SerialRead
==========

Read from a serial port

Properties
--------------

-   port (str): Serial port to read from
-   badurate (int): Baud rate of serial port

Dependencies
----------------

-   [pyserial](https://pypi.python.org/pypi/pyserial)

Commands
----------------
None

Input
-------
Any list of signals.

Output
---------
Same list of signals as input.
